Imagem

Estilizar imagens usando CSS

Imagens arredondadas

Use a border-radius para criar imagens arredondadas

Exemplo

img { border-radius: 10px;}
img { border-radius: 50%; }

Imagens responsivas

As imagens responsivas se ajustarão automaticamente ao tamanho da tela

Exemplo

.imgResp{
  max-width: 100%;
  height: auto;
}
