Sobre o HTML

HTML (Linguagem de Marcação de Hipertexto) é o código que você usa para estruturar uma página web e seu conteúdo. Por exemplo, o conteúdo pode ser estruturado em parágrafos, em uma lista com marcadores ou usando imagens e tabelas. Como o título sugere, este artigo fornecerá uma compreensão básica do HTML e suas funções.

Exemplos HTML básicos

Para criarmo uma pagina simples em html devemos especificar qual a versão que vamos utilizar, por exemplo esse site foi criado em HTML5.

Mas como indentificamos a versão que o site foi criado, é simples todo site estruturado em HTML5 possui em seu código a teg <!DOCTYPE HTML>.

Documentos HTML

Todos os documentos HTML devem começar com uma declaração de tipo de documento: .<!DOCTYPE html>

O documento HTML em si começa com e termina com <html></html>

A parte visível do documento HTML está entre <body></body>

Exemplo

<!DOCTYPE html>
<html>
<body>
<h1>My First Heading
</h1>
<p>My first paragraph.</p>
</body>
</html>
