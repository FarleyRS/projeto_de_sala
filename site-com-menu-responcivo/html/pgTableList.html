Tabelas e listas

Tabelas

As tabelas HTML permitem que os desenvolvedores web organizem dados em linhas e colunas.
As tegs que definem uma tabela são <tr> que são as linhas da colunas e <td> que são os dados.

Exemplo de tabela

Mateus Tobias Jessica
16 14 10

Codigo do exemplo

Exemplo de tabela

< table >
< tr >
< td > Email </ td >
< td > Tobias </ td >
< td > Linux </ td >
</ tr >
< tr >
< td > 16 </ td >
< td > 14 </ td >
< td > 10 </ td >
</ tr >
</ table >

Cabeçalho da tabela

Às vezes você quer que suas células sejam cabeçalhos, nesses casos use a tag < th > em vez da tag < td >

Exemplo de tabela

Pessoa 1 Pessoa 2 Pessoa 3
Mateus Tobias Jessica
16 14 10

Preenchimento e espaçamento de tabela

As tabelas HTML podem ajustar o preenchimento dentro das células e também o espaço entre as células

Preenchimento de células

Exemplo de preenchimento

th, td {
  padding: 15px;
}

Espaçamento entre células

O spacing é o espaço entre cada célula. Por padrão, o espaço é definido como 2 pixels.
Para alterar o espaço entre as células da tabela, use a border-spacing nas propriedades do CSS no table elemento:

Exemplo de espaçamento

table {
  border-spacing: 30px;
}
